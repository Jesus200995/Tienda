# Plan de Trabajo - El Comerciambre

## Información General
- Proyecto: El Comerciambre
- Tipo: Tienda en línea de ropa
- Frontend: Vue 3 + Vite + TailwindCSS + Pinia
- Backend: Laravel
- Base de Datos: MySQL
- Pagos: Mercado Pago
- Servidor: VPS Ubuntu + Nginx + SSL
